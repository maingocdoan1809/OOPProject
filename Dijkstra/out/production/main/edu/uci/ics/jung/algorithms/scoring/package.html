Mechanisms for assigning values (denoting significance, influence, centrality, etc.) to graph elements based on topological properties. These include:
  • BarycenterScorer: assigns a score to each node according to the sum of the distances to all other nodes
  • ClosenessCentrality: assigns a score to each node based on the mean distance to each other node
  • DegreeScorer: assigns a score to each node based on its degree
  • EigenvectorCentrality: assigns node scores based on long-term probabilities of random walks passing through the node at time t
  • PageRank: like EigenvectorCentrality, but with a constant probability of the random walk restarting at a uniform-randomly chosen node
  • PageRankWithPriors: like PageRank, but with a constant probability of the random walk restarting at a node drawn from an arbitrary distribution
  • HITS: assigns hubs-and-authorities scores to nodes based on complementary random walk processes
  • HITSWithPriors: analogous to HITS (see PageRankWithPriors)
  • VoltageScorer: assigns scores to nodes based on simulated current flow along edges
